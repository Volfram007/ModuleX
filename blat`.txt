Так как сломалась установка пакетов вот подсказка для восстановления.
Для настройки виртуального окружения (venv) в PyCharm внутри проекта, следуй этим шагам:
1. **Создание проекта (если еще не создан):**
   Если у тебя еще нет проекта, можно создать его через PyCharm. Если уже есть проект, переходи к следующему шагу.
2. **Открытие терминала в PyCharm:**
   В нижней части окна PyCharm нажми на вкладку `Terminal`, чтобы открыть встроенный терминал.
3. **Создание виртуального окружения:**
   В терминале введи следующие команды:
   ```sh
   python -m venv venv
   ```
   Здесь `venv` — это имя каталога, в котором будет создано виртуальное окружение. Ты можешь использовать любое другое имя, если хочешь.
4. **Активация виртуального окружения:**
   В зависимости от операционной системы используй соответствующую команду для активации виртуального окружения:
   - **Windows:**
     ```sh
     .\venv\Scripts\activate
     ```
5. **Установка необходимых пакетов:**
   После активации виртуального окружения можно устанавливать необходимые пакеты. Например:
   ```sh
   pip install <package_name>
   ```
6. **Настройка PyCharm для использования виртуального окружения:**
   После создания и активации виртуального окружения, нужно убедиться, что PyCharm использует его:
   - Открой `File` -> `Settings` (или `PyCharm` -> `Preferences` на macOS).
   - Перейди в раздел `Project: <Project Name>` -> `Python Interpreter`.
   - Нажми на иконку шестеренки и выбери `Add...`.
   - Выбери `Existing environment` и укажи путь к интерпретатору Python внутри виртуального окружения (например,
   `venv\Scripts\python.exe` для Windows или `venv/bin/python` для macOS/Linux).
   - Нажми `OK` и убедись, что новый интерпретатор выбран для твоего проекта.
Теперь виртуальное окружение настроено и используется в твоем проекте в PyCharm.

Чтобы установить пакеты из файла `requirements.txt` в виртуальное окружение, выполните следующие шаги:
1. **Убедитесь, что виртуальное окружение активировано:**
   - **Windows:**
     ```sh
     .\venv\Scripts\activate
     ```
   - **macOS/Linux:**
     ```sh
     source venv/bin/activate
     ```
2. **Установка пакетов из `requirements.txt`:**
   Выполните следующую команду в терминале PyCharm:
   ```sh
   pip install -r requirements.txt
   ```
Файл `requirements.txt` должен находиться в корне вашего проекта и содержать список всех необходимых пакетов с указанием их версий (если нужно).
Эта команда установит все перечисленные пакеты в активированное виртуальное окружение.
